# 555-Timer-Oscillator-PCB

This project is a PCB designed in KiCad implementing a 555 timer in astable mode to alternate two LED outputs.

## Design Overview
- Designed schematic and PCB layout using KiCad
- Selected resistor and capacitor values to control oscillation frequency
- Implemented alternating LED outputs using a 555 timer IC

## Design Feature
A key design detail was keeping the layout compact and minimizing trace length between timing components. This helps reduce noise sensitivity and improves signal stability in the oscillator circuit.

## What I Learned
- Translating schematics into manufacturable PCB layouts
- Component placement and routing tradeoffs
- Debugging and validating hardware after fabrication
